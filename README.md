# ✳️ Save A Class

Save a class is a app that grabs your Modern Warfare(2019) match history data, so you can save your best loadouts!

## ✔️ Tech

- Nextjs
- [call-of-duty-api](https://www.npmjs.com/package/call-of-duty-api)
- MongoDB
- Tailwindcss
- Typescript
- Authentication (jwt, bcrypt)

## ✔️ Functionality

- Create account
- Authentication
- Save data to DB
- Delete data from DB

## ✔️Install

```
yarn install
```

### ✔️ Develop

```
yarn dev
```
